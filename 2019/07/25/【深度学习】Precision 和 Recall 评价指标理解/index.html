йў„жөӢз»“жһңдёәйҳіжҖ§ Positive йў„жөӢз»“жһңдёәеҒҮйҳіжҖ§ Negative
йў„жөӢз»“жһңжҳҜзңҹе®һзҡ„ True TP TN
йў„жөӢз»“жһңжҳҜиҷҡеҒҮзҡ„ False FP FN

йҖҡеёёе…іжіЁзҡ„зұ»дёәжӯЈзұ»пјҢе…¶д»–зұ»дёәиҙҹзұ»гҖӮпјҲд»ҘзҢ«зӢ—дәҢеҲҶзұ»дёәдҫӢпјҢзҺ°еңЁе…іжіЁзӢ—зҡ„precisionе’Ңrecallпјү

TP жӯЈзұ»йў„жөӢдёәжӯЈзұ»пјҲйў„жөӢеҮәзӢ—зҡ„еӣҫзүҮе®һйҷ…ж ҮжіЁд№ҹжҳҜзӢ—пјү
FN жӯЈзұ»йў„жөӢдёәиҙҹзұ»пјҲйў„жөӢеҮәзҢ«зҡ„еӣҫзүҮе®һйҷ…ж ҮжіЁжҳҜзӢ—пјү
FP иҙҹзұ»йў„жөӢдёәжӯЈзұ»пјҲйў„жөӢеҮәзӢ—зҡ„еӣҫзүҮе®һйҷ…ж ҮжіЁжҳҜзҢ«пјү
TN иҙҹзұ»йў„жөӢдёәиҙҹзұ»пјҲйў„жөӢеҮәзҢ«зҡ„еӣҫзүҮе®һйҷ…ж ҮжіЁжҳҜзҢ«пјү

2. Precision

зІҫзЎ®зҺҮи®Ўз®—е…¬ејҸпјҡ

P=TPTP+FPP=\frac{TP}{TP+FP} P=TP+FPTPвҖӢ

зҗҶи§Јпјҡ

TP+FP: д№ҹе°ұжҳҜе…ЁдҪ“PositiveпјҢ д№ҹе°ұжҳҜйў„жөӢзҡ„еӣҫзүҮдёӯжҳҜжӯЈзұ»зҡ„еӣҫзүҮзҡ„ж•°зӣ®

TP: д№ҹе°ұжҳҜжӯЈзұ»д№ҹиў«йў„жөӢдёәжӯЈзұ»зҡ„еӣҫзүҮзҡ„дёӘж•°

жҖ»д№Ӣпјҡйў„жөӢжӯЈзЎ®зҡ„еӣҫзүҮдёӘж•°еҚ жҖ»зҡ„жӯЈзұ»йў„жөӢдёӘж•°зҡ„жҜ”дҫӢпјҲд»Һйў„жөӢз»“жһңи§’еәҰзңӢпјҢжңүеӨҡе°‘йў„жөӢжҳҜеҮҶзЎ®зҡ„пјү

3. Recall

еҸ¬еӣһзҺҮи®Ўз®—е…¬ејҸпјҡ

R=TPTP+FNR=\frac{TP}{TP+FN} R=TP+FNTPвҖӢ

зҗҶи§Јпјҡ

TP+FN: д№ҹе°ұжҳҜе…ЁдҪ“е®Ңе…Ёж»Ўи¶іеӣҫзүҮж ҮжіЁзҡ„еӣҫзүҮзҡ„дёӘж•°

TPпјҡжӯЈзұ»иў«йў„жөӢдёәжӯЈзұ»зҡ„еӣҫзүҮдёӘж•°

жҖ»д№ӢпјҡзЎ®е®ҡдәҶжӯЈзұ»иў«йў„жөӢдёәжӯЈзұ»еӣҫзүҮеҚ жүҖжңүж ҮжіЁеӣҫзүҮзҡ„дёӘж•°пјҲд»Һж ҮжіЁи§’еәҰзңӢпјҢжңүеӨҡе°‘иў«еҸ¬еӣһпјү
